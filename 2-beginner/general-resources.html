General Resources

Online courses

→ All resources are (mostly) free unless stated otherwise.

Codecademy: Learn anything from Python to CSS to SQL! Codecademy has lessons you can follow so that you can learn how to code without knowing anything about variables, classes, or whatever else you might need to know.

Coursera:

Udemy: [PAID but there are free coding courses available]

The Odin Project: Looking to learn about web development? The Odin Project is an open-source resource that eases you into projects so that you can learn to code and create something on your own.

Microsoft Learn Resources:

  • includes a bunch of step-by-step tutorials on trendy topics like Cloud, AI, ML to make simple yet cool projects
  • has additional links to teaching beginners the basics of python and C#
